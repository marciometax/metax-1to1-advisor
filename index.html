🔵 Configurar Microsoft 365
Para envio direto sem abrir o Outlook, registre um App no Azure Active Directory e cole as credenciais abaixo. O sistema usará OAuth 2.0 para enviar e-mails via Microsoft Graph API.
Como obter:
1. Acesse portal.azure.com → Azure Active Directory → Registros de Aplicativo
2. Novo registro → nome: MTX Advisor → Contas deste diretório
3. URI de redirecionamento: https://mtxadvisor.netlify.app
4. Permissões de API: Mail.Send · Calendars.ReadWrite · User.Read
5. Copie o ID do Aplicativo (cliente) e o ID do Diretório (locatário)
